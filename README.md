# WebOS DeskFlow v2.0

A merged production build of WebOS DeskFlow, featuring a Zustand WindowManager, IPC System Bus, and a Virtual File System.

## Features
- **Kernel**: Zustand WindowManager + IPC System Bus + Virtual File System
- **Intelligence**: Cortex AI with real OS control
- **Apps**: 12 fully functional applications
- **Architecture**: src/core + src/store + src/apps + src/desktop

## Getting Started
This repository contains the single-file production build `App.jsx`. You can integrate this into your React project to run the WebOS environment.
